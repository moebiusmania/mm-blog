Christmas Special - Pack 2009

Iniziamo la settimana "special" dedicata al Natale con un bel regalo... il Christmas Pack 2009 di MoebiusMania.net! Per chi è nuovo alla cosa, è un file .zip gratuito da scaricare con del materiale per (letteralmente) addobbare il vostro sito o il vostro blog, all'interno troverete: varie imagini in formato Png, un file Css con gli stili necessari per applicare le decorazioni alla vostra pagina web, un file .ai da aprire con Illustrator o Inkscape per modificare e personalizzare le grafiche vettoriali, un file Html di esempio e un file con le istruzioni. La parte grafica include alcune mie piccole realizzazioni e alcuni file gratuiti reperiti nel web leggermente modificati. mentre il Css e tutto il resto è farina del mio sacco. Ovviamente potete sia usare i Css scritti nel file allegato anche per altri progetti sia sostituire le immagini , e spero facciate così inserendo qualche vostra realizzazione.
