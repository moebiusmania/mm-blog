Un buon inizio 2013 per jQuery

Giornate interessanti per i molti utilizzatori della libreria jQuery! In poche ore sono ci sono state diverse novità, un elenco veloce:
  • E' uscito jQuery 1.9 , l'ultima versione a supportare Internet Explorer versioni 6, 7 e 8
  • Disponibile la prima beta di jQuery 2.0
  • Finalmente ha riaperto il sito-directory dei plugin, con una nuova veste grafica
  • E' stato rilasciato il plugin Migrate alla prima versione
  • Rilasciato jQuery UI 1.10
Più nel dettaglio cosa c'è da sapere?jQuery 1.9 principalmente ha deprecato diverse vecchie API già segnalate obsolete dalla versione 1.8, da adesso in poi se avrete bisogno di retro-compatibilità per Internet Explorer questa sarà l'ultima versione a supportarla (anche in futuro). Proprio per aiutare il passaggio dalla versione 1.8 alla 1.9 è stato creato il plugin Migrate, che permette di utilizzare ancora le API deprecate con l'ultima release della libreria. Stando ai piani del team di sviluppo a breve vedremo jQuery 2.0 che come ho spiegato in un precedente articolo sarà molto più leggero e veloce dedicandosi principalmente ai browser moderni, se non potete aspettare testate questa beta! Dopo vari mesi offline per motivi tecnici torna il sito dei plugin gestito dalla community di jQuery, il suo scopo è quello di creare un punto centrale di riferimento per la ricerca e la condivisione dei plugin che sicuramente semplificherà la vita a molti, sia per chi crea plugin sia per chi li deve usare. Insieme alla 1.9 è stata rilasciata la versione 1.10 di jQuery UI, la nota suite di componenti per creare interfacce, che introduce la compatibilità con l'ultima versione di jQuery, viene deprecato il supporto per la versione 1.8 (potete sempre scaricare jQuery UI 1.9.2 per lavorare con la vecchia versione) e vede il redesign delle API per alcuni componenti, quindi date un occhio alla documentazione se lo avete già usato in passato! Per maggiori dettagli e le note ufficiali potete consultare il blog ufficiale del team di jQuery.
